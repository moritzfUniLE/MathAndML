About Synthetic Generation
Graph Types:
  • Erdős–Rényi (ER): Random connections between nodes
  • Scale-Free (SF): Power-law degree distribution (hubs)
  • Bipartite (BP): Hierarchical two-layer structure
Benefits:
  • Known ground truth for evaluation
  • Controlled complexity testing
  • Algorithm benchmarking
  • Parameter sensitivity analysis
